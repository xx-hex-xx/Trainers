[Changes_EN]***Update: 08/25/2025 (version 1.5)***
The trainer structure has been changed, the configuration files will be recreated.
– Added:
◦ The "Playing without gas mask" function;
◦ The "Freeze everyone in place" function;
◦ The "Turn off lights" function.
– Changed:
◦ The "Invulnerability" function. Now there is no animation from damage.
– Fixed:
◦ The "Infinite ammo" function. Sometimes after reloading the amount of ammo increased.

***Update: 08/24/2024 (version 1.4)***
– Fixed:
◦ Crash when finding game process on Steam version due to updated GameOverlayRenderer64.dll file.

***Update: 07/03/2024 (version 1.3)***
– Fixed:
◦ No workability the "Infinite medkits" function on game version 2.0.0.1 (Enhanced Edition).
– Changed:
◦ Method for disabling the exclusive input mode for the keyboard (DirectInput::SetCooperativeLevel) in dinput8.dll. Now the trainer does not make changes to this system library but works specifically with the game so there should no longer be problems when the keyboard and/or mouse might stop working when starting the trainer and switching to the game.

***Update: 06/17/2024 (version 1.2)***
– Fixed:
◦ Method for disabling exclusive input mode (DirectInput::ExclusiveCooperativeLevel) in dinput8.dll. If you have two or more monitors, the mouse cursor will no longer go outside the game window.
– Added:
◦ Possibility of autoloading of saved functions when finding a game process. To enable this option go to the trainer settings window (disabled by default);
◦ Notifications during autoload as well as when manually saving/loading saved functions. You can disable them by selecting "Don't show again" in the notification.
Changed:
◦ Library for calling the sound of activation/deactivation of functions due to which the trainer could not start on Windows 7.

***Update: 02/27/2024 (version 1.1)***
Added support for Enhanced Edition.
– Fixed:
◦ A bug in the "Infinite air bottle pressure" function which could lead to a crash from the game.
– Added:
◦ Function "Infinite "Hellbreath" battery charge";
◦ Function "No clip mode".
– Changed:
◦ In the functions "Infinite throwing knives", "Infinite decoys", " Infinite hand grenades" and "Infinite Molotov cocktails" do not decrease the current number of items;
◦ The "Time of day" function is divided into two functions: "Increase time of day (+1 hour)" and "Decrease time of day (-1 hour)" ;
◦ The "Infinite air bottle pressure" function also affects the flamethrower;
◦ The "Infinite flashlight battery charge" function does not reduce the current amount of charge.[Changes]

◦ Функция "Отключить освещение".
– Изменено:
◦ Функция "Неуязвимость". Теперь нет анимации от урона.

◦ Функция "Бесконечные боеприпасы". Иногда после перезарядки количество боеприпасов увеличивалось.

***Обновление: 24.08.2024 (версия 1.4)***
– Исправлено:
◦ Вылет при нахождении процесса игры на Steam-версии из-за обновленного файла GameOverlayRenderer64.dll.

***Обновление: 03.07.2024 (версия 1.3)***
– Исправлено:
◦ Неработоспособность функции "Бесконечные аптечки" на версии игры 2.0.0.1 (Enhanced Edition).
– Изменено:
◦ Способ отключения эксклюзивного режима ввода для клавиатуры (DirectInput::SetCooperativeLevel) в dinput8.dll. Теперь трейнер не вносит изменения в эту системную библиотеку, а работает именно с игрой, поэтому больше не должно возникнуть проблем, когда при запуске трейнера и переключения на игру могли перестать работать клавиатура и/или мышь.

***Обновление: 17.06.2024 (версия 1.2)***
– Исправлено:
◦ Способ отключения эксклюзивного режима ввода (DirectInput::ExclusiveCooperativeLevel) в dinput8.dll. Если у вас два и более монитора, то курсор мыши больше не будет выходить за пределы окна игры.
– Добавлено:
◦ Возможность автозагрузки сохраненных функций при нахождении процесса игры. Для включения этого параметра перейдите в окно настроек трейнера (по умолчанию – отключен);
◦ Уведомления при автозагрузке, а также при ручном сохранении/загрузке сохраненных функций. Вы можете отключить их, выбрав в уведомлении "Больше не показывать".
– Изменено:
◦ Библиотека вызова проигрывания звука активации/деактивации функций, из-за которой трейнер мог не запуститься на Windows 7.

***Обновление: 27.02.2024 (версия 1.1)***
Добавлена поддержка Enhanced Edition.
– Исправлено:
◦ Ошибка в функции "Бесконечное давление в баллоне", которая могла привести к вылету из игры.
– Добавлено:
◦ Функция "Бесконечный заряд батареи "Рельсы"";
◦ Функция "Режим "No clip"".
– Изменено:
◦ В функциях "Бесконечные метательные ножи", "Бесконечные приманки", "Бесконечные гранаты" и "Бесконечные коктейли Молотова" не уменьшается текущее количество предметов;
◦ Функция "Время суток" разделена на две функции: "Увеличить время суток (+1 час)" и "Уменьшить время суток (-1 час)";
◦ Функция "Бесконечное давление в баллоне" влияет и на огнемет;
◦ В функции "Бесконечный заряд батареи фонаря" не уменьшается текущее количество заряда.[Changes]
