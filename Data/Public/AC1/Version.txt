[Changes_EN]***Update: 04/07/2025 (version 1.7)***
The trainer structure has been changed, the configuration files will be recreated.
– Changed:
◦ The "No clip mode" function. Due to the instability of the previous version of the No clip mode it was reworked and now works like in trainers for other titles of the Assassin's Creed series.

***Update: 10/25/2024 (version 1.6)***
– Changed:
◦ The "Invulnerability" function. No damage animation.

***Update: 09/17/2024 (version 1.5)***
– Fixed:
◦ Bug in the "Fast movement speed" function due to which the Left Ctrl and Space keys could be disabled when using the "No clip mode" function.
– Added:
◦ The "Freeze everyone in place" function.
– Changed:
◦ The "Invisibility" function has been reworked. The player is not noticed by guards standing guard, archers on the roof, guards rescuing townspeople and drunkards;
◦ The "Fast movement speed" function affects the player when he climbs the stairs;
◦ The "Teleport to custom marker" function has been reworked. The teleport works stably and no longer throws the player upward in some cases.

***Update: 07/27/2024 (version 1.4)***
The trainer has been completely reworked.
– Fixed:
◦ Bug in the "Invulnerability" function due to which enemies did not receive damage when falling;
◦ Bug in the "Invisibility" function due to which eavesdropping quests could become impassable;
◦ Bug in the "Teleport to custom marker" function due to which the teleport could not work in some memory sequences.
– Added:
◦ The "No clip mode" function.
– Changed:
◦ The "Stop countdown timer" function. When you turn off the function the timer will immediately resume instead of remaining frozen until the game is minimized;
◦ The "Fast movement speed" function also affects Desmond;
◦ The "Teleport to custom marker" function. Teleport also works using icons on the map.

***Update: 05/29/2022 (version 1.3)***
– Fixed:
◦ Bug with text display on high-resolution monitors.
– Changed:
◦ General optimization of the trainer.

***Update: 05/29/2022 (version 1.2)***
– Fixed:
◦ Bug with the Delete hotkey.

***Update: 05/19/2022 (version 1.1)***
– Changed:
◦ The "Invulnerability" function.[Changes]
